## ⚠️ Warning
The lock screen feature may only work on **Windows 11 Pro**, and may have issues on some devices.  
This script was originally created as a PowerShell learning project, and after testing, it works as intended (from the creator).
---
## 📖 How to Use
1. Install PowerShell  
   🔗 [Microsoft Official Guide](https://learn.microsoft.com/en-us/powershell/scripting/install/installing-powershell-on-windows?view=powershell-7.5)
2. Run `theme-switcher.bat` as **Administrator**
3. Select the desired mode:
   - `0` — Dark Mode  
   - `1` — Light Mode
4. Press Enter and wait for the script to complete.
5. ✅ Done!
---
## 🖼️ How to Use Your Own Wallpaper
Replace the files in the `/wallpaper/` folder with your own images:
- For **dark mode**:  
  Rename your image to `ROG G15.png` (must be `.png`) and replace the existing one.
- For **light mode**:  
  Rename your image to `ROGDARE.jpg` (must be `.jpg`) and replace the existing one.
📂 Folder: `~\Wallpaper.theme\wallpaper`
---

## 📖 วิธีใช้งาน
1. ติดตั้ง PowerShell  
   🔗 [คู่มือการติดตั้ง (จาก Microsoft)](https://learn.microsoft.com/en-us/powershell/scripting/install/installing-powershell-on-windows?view=powershell-7.5)
2. เปิดไฟล์ `theme-switcher.bat` ด้วยสิทธิ์ **ผู้ดูแลระบบ (Run as Administrator)**
3. เลือกโหมดที่ต้องการ:
   - `0` — โหมดมืด  
   - `1` — โหมดสว่าง
4. กด Enter แล้วรอให้โปรแกรมทำงานจนเสร็จ
5. ✅ เสร็จเรียบร้อย!
---
## 🖼️ วิธีเปลี่ยนภาพพื้นหลังเป็นของคุณเอง
ให้นำไฟล์ภาพของคุณมาแทนที่ในโฟลเดอร์ `/wallpaper/` โดย:
- **ธีมมืด (Dark Mode)**:  
  เปลี่ยนชื่อไฟล์ภาพเป็น `ROG G15.png` (ต้องเป็น `.png`) แล้วแทนที่ไฟล์เดิม
- **ธีมสว่าง (Light Mode)**:  
  เปลี่ยนชื่อไฟล์ภาพเป็น `ROGDARE.jpg` (ต้องเป็น `.jpg`) แล้วแทนที่ไฟล์เดิม
📂 โฟลเดอร์: `~\Wallpaper.theme\wallpaper`
