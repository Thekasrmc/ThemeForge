ความต้องการของระบบ
	Windows 10 หรือ Windows 11
	PowerShell
 	ต้องรันในโหมด Administrator
วิธีใช้งาน
	เปิดโฟลเดอร์ ThemeForge
	ดับเบิลคลิกไฟล์ theme-switcher.bat
	กด Yes เมื่อ Windows ขอสิทธิ์ Administrator
เลือกตัวเลือก
	0 = Dark Mode
	1 = Light Mode
	กด Enter แล้วรอให้โปรแกรมทำงานจนเสร็จ
	เมื่อเสร็จสิ้น โปรแกรมจะแจ้งสถานะและปิดหน้าต่างอัตโนมัติ
การตั้งค่าภาพพื้นหลัง
	ให้นำไฟล์ภาพไปวางไว้ในโฟลเดอร์
	ThemeForge\wallpaper
	ตั้งชื่อไฟล์ดังนี้
		Light Mode	Dark Mode
		Light.png	Dark.png
		Light.jpg	Dark.jpg

ระบบจะเลือกไฟล์ .PNG ก่อน หากไม่มีไฟล์ .PNG จึงจะเลือก .JPG

หมายเหตุเกี่ยวกับ Lock Screen
ฟีเจอร์ Lock Screen อาจใช้งานได้ในบางระบบ เช่น Windows 10 หรือ Windows 11 ทั้งรุ่น Home และ Pro ทั้งนี้ขึ้นอยู่กับการตั้งค่าของระบบและสิทธิ์ของเครื่อง หากตั้งค่าไม่สำเร็จ ผู้ใช้สามารถตั้งค่า Lock Screen ผ่านเมนู Windows Settings ได้ด้วยตนเอง เเล้วกลับมาใช้โปรเเกรมนี้เพื่อเปลี่ยนอีกครั้ง

คำเตือน
การทำงานของโปรแกรมนี้มีการปรับแต่งค่าระบบบางส่วน แม้จะผ่านการทดสอบโดยผู้จัดทำแล้ว (Windows 11 Pro) และไม่พบปัญหาใด ๆ แต่ผู้พัฒนาไม่รับผิดชอบต่อความเสียหายที่อาจเกิดขึ้นกับระบบของคุณ
เมื่อคุณใช้งานโปรแกรมนี้ ถือว่าคุณยอมรับข้อตกลงดังกล่าวแล้ว
